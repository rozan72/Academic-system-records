# Academic-system-records
The Academic Record Management System is a web-based application developed using HTML5, CSS3, JavaScript, PHP, and MySQL to help manage student academic information efficiently. The system allows administrators to add and view student records, manage courses, assign grades, and maintain academic data through a simple and responsive user interface. It uses PHP for server-side processing, MySQL for database storage, and JavaScript for dynamic interactions. To run the project, install XAMPP, start Apache and MySQL, place the project folder inside `C:\xampp\htdocs\academic-system`, create a database named `academic_system` in phpMyAdmin, import the provided `db.sql` file, and access the application through `http://localhost/academic-system/` or `http://localhost:8080/academic-system/` if Apache is configured on port 8080. The project is intended for educational purposes and demonstrates CRUD operations, database connectivity, and basic academic record management. Future enhancements may include user authentication, role-based access control, GPA calculation, transcript generation, PDF reports, advanced search and filtering, dashboard analytics, and mobile responsiveness. This project serves as a practical example of full-stack web development using PHP and MySQL and can be extended into a complete university or school management system.
